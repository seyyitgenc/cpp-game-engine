# this project is all about SDL based c++ game engine

> REQUIRED LIBS FOR OpenGL branch

- OpenGL, GLFW, GLM, Assimp, GLEW  

## Follow these steps

    mkdir build
    cd build
    cmake .. -G "Unix Makefiles"
    make

- Change this -G parameter according to your build preferences
